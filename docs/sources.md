# Description of Data Sources

All data collected from the [California Statewide Database](https://statewidedatabase.org/) and the [US Census Bureau](https://www.census.gov/en.html)

### Voting Data

- [Election Data by SRPrecinct](https://statewidedatabase.org/election.html) - [Download](https://statewidedatabase.org/pub/data/G22/state/state_g22_sov_data_by_g22_srprec.zip)
- [Precint ot Block conversion file](https://statewidedatabase.org/d20/g22_geo_conv.html) - [Download](https://statewidedatabase.org/pub/data/G22/state/state_g22_sr_blk_map.csv)

### Census Data

- [2020 population data](https://statewidedatabase.org/d20/2024_cvap.html)- [Download](https://statewidedatabase.org/ftp/pub/data/CVAP_2024/state/state_CVAP_with_DOJ_2018-2022_by_2020_Blocks.csv)
- [2010 population data](https://statewidedatabase.org/d10/2020_cvap.html) - [Download](https://statewidedatabase.org/ftp/pub/data/CVAP_2020/Total_cvap_by_block.csv)

### Shapefiles

- [2020 Block Group Shapefile](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/) - [Download](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/tl_2020_06_bg20.zip)
- [2010 Block Group Shapefile](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/) - [Download](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/tl_2020_06_bg10.zip)
