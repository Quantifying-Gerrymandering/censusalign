# Description of Data Sources

All data collected from the [California Statewide Database](https://statewidedatabase.org/) and the [US Census Bureau](https://www.census.gov/en.html)

### Voting Data

- [Election Data by SRPrecinct](https://statewidedatabase.org/election.html)
  - [Download 2022](https://statewidedatabase.org/pub/data/G22/state/state_g22_sov_data_by_g22_srprec.zip)
- [Precinct to Block Conversion File](https://statewidedatabase.org/d20/g22_geo_conv.html)

### Census Data

- [2020 population data](https://statewidedatabase.org/d20/2024_cvap.html)
- [2010 population data](https://statewidedatabase.org/d10/2020_cvap.html)

### Shapefiles

- [2020 Block Group Shapefile](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/)
- [2010 Block Group Shapefile](https://www2.census.gov/geo/tiger/TIGER2020PL/STATE/06_CALIFORNIA/06/)
